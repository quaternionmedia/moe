Dev
-------
install venv
`sudo apt-get install python3-venv`

create venv
`python3 -m venv moenv` (change `moenv` to your desired environment name)

activate
`source moenv/bin/activate`

install dev requirements
`pip install -r requirements_dev.txt`

Coverage
-------
`make coverage`
