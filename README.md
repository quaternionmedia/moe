# M.O.E.
Meticulous Oversight Engine

### install
`git clone git@github.com:quaternionmedia/moe.git`

`pip install -e moe`

`cd moe/`

`./moe.sh init`


### run

`./moe.sh run`

If you need to rebuild the docker images:

`./moe.sh run --build`

... then open a browser at [localhost:5000](http://localhost:5000)
